Feedstock for [epic](https://github.com/eic/epic) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

DD4hep Geometry Description of the ePIC Experiment

## Install

```bash
conda install -c hep-forge -c conda-forge epic
```
